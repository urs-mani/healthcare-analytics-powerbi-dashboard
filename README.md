# Healthcare Analytics Dashboard (Power BI)

## Project Overview
This project analyzes hospital patient data using Microsoft Power BI to provide insights into patient admissions, hospital revenue, medical conditions, and doctor performance.

## Dashboard Pages
1. Hospital Overview
2. Patient Analytics
3. Hospital Insights
4. Advanced Analytics

## Key Features
KPI Cards (Total Patients, Revenue, Avg Age)
Patient Admission Trends
Medical Condition Analysis
Revenue Breakdown
Year-over-Year Growth Analysis
Decomposition Tree Visualization

## Tools Used
Microsoft Power BI
DAX (Data Analysis Expressions)
Data Visualization

## Dataset
Healthcare patient dataset including hospital, doctor, admission dates, billing amount, and medical conditions.
